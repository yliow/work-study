The first thing you do is you go into the assignment_maker.py and edit the name of the class, the questions list, the author, the assignment number and the desired directory you want your assignment to go. Then you start adding the link to the directory of the new assignment format the different documents type are as follows

a.add_programming_question() adding question has a programming question as the name suggest
a.add_math_question() adding question that require a solution file
a.add_skeleton() adds a skeleton code file for previous assignment
a.add_latexstr() adds a specific latex string to the main.tex
a.add_extra_document() documents that would be in the main directory
