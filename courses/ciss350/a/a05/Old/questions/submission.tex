\begin{Verbatim}[fontsize=\footnotesize]
...
  a

      |
      +-- main.tex (and other LaTeX files)
      |
      +-- a05q01
      |   |
      |   +-- main.cpp
      |
      +-- a05q02
          |
          +-- main.cpp
\end{Verbatim}
When you're done, go to \verb!ciss350/a/!
create \verb!a05.tar.gz! (or \verb!submit.tar.gz!) that holds
the \verb!a05! directory as usual.
