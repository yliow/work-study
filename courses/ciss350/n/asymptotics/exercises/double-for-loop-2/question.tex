The following algorithm
computes the sum of upper triangular 
values in a 2D array $x$ of size 
$n$-by-$n$:
\begin{Verbatim}[frame=single, fontsize=\small]
s = 0
for i = 0, 1, 2, ..., n - 1:
    for j = i, i + 1, i + 2, ..., n - 1:
        s = s + x[i][j]
\end{Verbatim}
Compute the big-O of the runtime of the above algorithm.
