There are two possible outermost for-loop: 

Either
\begin{verbatim}
        for i = 1, 2, ..., 12
            [some code to draw 1 line]
\end{verbatim}

or
\begin{verbatim}
        for i = 1, 2, 3, 4:
            [some code to draw 3 lines]
\end{verbatim}
