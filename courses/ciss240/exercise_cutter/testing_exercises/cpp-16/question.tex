Try this:
\begin{console}
#include <iostream>

int main()
{
    std :: cout << "Hello, world!\n";

    return 0;
}
\end{console}
Does it work?
Now try this:
\begin{console}
#include <iostream>

int main()
{
    std:     :cout << "Hello, world!\n";

    return 0;
}
\end{console}
Does it work?
