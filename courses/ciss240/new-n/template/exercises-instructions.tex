\end{python0}

To create an exercise, in your bash shell, do
\begin{console}
cd exercises
solutions make-ex exercise-000
cd ..
\end{console}
(I assume there's an \verb!exercises! directory in the
directory containing the \LaTeX\ files for your chapter.
If not, create one.)
Then you'll get \verb!exercises/exercise-000! directory
with four files:
\begin{console}
answer.tex  main.tex  makefile  question.tex
\end{console}

Open \verb!exercises/exercise-000/question.tex! and enter your question:
\begin{console}
\tinysidebar{\debug{exercises/{exercise-000/question.tex}}}
What is $1 + 1$?
\end{console}
Do \textit{not} change the first line.
You can enter your answer in
\verb!exercises/exercise-000/answer.tex!.

In your \LaTeX\ file where you want this question to appear,
enter
\begin{console}
\input{exercises/exercise-000/main.tex}
\end{console}
