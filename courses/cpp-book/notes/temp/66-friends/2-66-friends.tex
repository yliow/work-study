\sectionthree{Friend member function}

You can also make a member function of a different class, i.e., a method
of a different class, a friend of the current class.

For example:

\begin{consolethree}[escapeinside=||]
class C
{
public:
    ...
    void f(C c);
    ...
};

class D
{
public:
    ...
    friend void C::f(C);
    ...
};
\end{consolethree}
