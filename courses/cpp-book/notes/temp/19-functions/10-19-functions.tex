\sectionthree{Testing}

Not only is it easier to reuse your code when you broke up your program into functions, it's also easier to test your code.

For instance suppose you have written your function to compute max of three integer values. You can write a function \texttt{test\_max3()} that call \texttt{max3()} with specific values and verifies that the return values are correct:

\begin{consolethree}[escapeinside=||]
void test_max3()
{
    // max of 1, 2, 3 should be 3
    if (max3(1, 2, 3) != 3)
    {
        std::cout << "test failed: max3(1,2,3) should return 3\n";
    }
    // repeat for other test cases
}
\end{consolethree}

Primes are used a lot in cryptography and computer security. So perhaps you want to make sure your \texttt{isprime()} is absolutely correct:

\begin{consolethree}[escapeinside=||]
void test_isprime()
{
    if (isprime(2) != true)
    {
        std::cout << "test failed: isprime(2) should be true\n";
    }
    if (isprime(3) != true)
    {
        std::cout << "test failed: isprime(3) should be true\n";
    }
    if (isprime(4) != false)
    {
        std::cout << "test failed: isprime(4) should be false\n";
    }
    // ... more test cases
}
\end{consolethree}
