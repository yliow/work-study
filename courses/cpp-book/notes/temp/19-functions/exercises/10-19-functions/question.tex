\begin{ex}
Invent your own ASCII art problem (or take the code from earlier notes or assignments and write a function for it) and write a function for it. Here's one: Write the function \texttt{drawSquare()} that accepts an int value and draw a box with height and width of the specified \texttt{int} length. For instance calling \texttt{drawSquare(3)} prints this to the console window:

\begin{console}
***
* *
***
\end{console}

and calling \texttt{drawSquare(8)} prints this:

\begin{console}
********
*      *
*      *
*      *
*      *
*      *
*      *
********
\end{console}
\end{ex}
