\begin{ex}
Write a function \texttt{sum()} that accepts an integer value and returns the sum of all integer values from 0 to the given integer value. For instance \texttt{sum(0)} returns 0; \texttt{sum(3)} returns 6 (since 0 + 1 + 2 + 3 gives 6).

Here's something that might help:

\begin{consolethree}[escapeinside=||]
#include <iostream>

int sum(int n)
{
    int s = 0;
    // write a for-loop so that s = 0 + 1 + ... + n
    return s;
}

int main()
{
    for (int i = 0; 5; ++i)
    {
        std::cout << sum(i) << '\n';
    }
    return 0;
}
\end{consolethree}
\end{ex}
