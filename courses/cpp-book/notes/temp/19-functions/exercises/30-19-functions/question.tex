\begin{ex}
Write a function \texttt{digit()} such that \texttt{digit(n, i)} returns the \texttt{i}-th digit of \texttt{n}. For instance \texttt{digit(834, 0)} return \texttt{4}, \texttt{digit(834, 1)} returns \texttt{3}, \texttt{digit(834, 2)} returns \texttt{8}, and \texttt{digit(834, 3)} returns \texttt{0}.
\end{ex}
