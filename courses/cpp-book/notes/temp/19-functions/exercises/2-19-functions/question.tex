\begin{ex}
Write a function \texttt{sign()} that accepts a \texttt{double} and returns 1 if the \texttt{double} is > 0, 0 if the \texttt{double} is 0, or -1 if the \texttt{double} is < 0.

Note in this case that the return value is an \texttt{int} (yes you can use a \texttt{double} but that would be an overkill -- not to mention waste of memory since a \texttt{double} takes up more space.)

Here's skeleton for you:

\begin{consolethree}[escapeinside=||]
int sign(double x)
{
    ...
}
\end{consolethree}

Of course you should always test your code.
\end{ex}
