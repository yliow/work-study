\sectionthree{Global variables and global constants}

Based on the way C++ search for a variable, can you figure out the output of the following program? Verify your answer.

\begin{consolethree}[escapeinside=||]
#include <iostream>

int y = 1;

void f()
{
    int y = 42;
    std::cout << y << '\n';
}

int main()
{
    std::cout << y << '\n';
    f();
    std::cout << y << '\n';
    return 0;
}
\end{consolethree}

Using only parameters and return values, you can tell which exactly what information is going in and out of a function. That's a big advantage. The moment you start adding global variables everywhere, things become extremely messy.
