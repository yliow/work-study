\sectionthree{Scope}

You have talked about scope before. The same thing applies to functions:

\begin{consolethree}[escapeinside=||]
void f(int x)
{
    int y = 0;
    std::cout << x << ' ' << y << '\n';
    return;
}

int main()
{
    int y = 42;
    f(y); // what is the value of the parameter x?
    std::cout << y << '\n';
    return 0;
}
\end{consolethree}

Although you can nest for-loops to get double for-loops, you can nest if
statements, etc., you \textbf{CANNOT} nest functions:

\begin{consolethree}[escapeinside=||]
// THIS CODE WILL NOT COMPILE
#include <iostream>

void f()
{
    void g() // ILLEGAL!!!
    {
        ...
    }
    ...
}

int main()
{
    f();
    return 0;
}
\end{consolethree}
