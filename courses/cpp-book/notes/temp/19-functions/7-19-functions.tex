\sectionthree{Functions with two parameters}

The following function has two parameters:

\begin{consolethree}[escapeinside=||]
#include <iostream>

int max2(int x, int y)
{
    if (x >= y)
    {
        return x;
    }
    else
    {
        return y;
    }
}

int main()
{
    std::cout << max2(3, 5) << '\n';
    return 0;
}
\end{consolethree}

Just like the for-loop and while-loop, you should NOT have too many arguments. If you have more than 10 parameters you should step back and make sure your design is correct.

Note only that, the parameters need not have the same types. Something like this is OK:

\begin{consolethree}[escapeinside=||]
int f(int x, double y, char z)
{
    ...
}
\end{consolethree}
