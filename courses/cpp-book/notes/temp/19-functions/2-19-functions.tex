\sectionthree{Pass-by-Value}

It's extremely important to remember that \texttt{main()} calls \texttt{square()} by passing in a value. This is called \textbf{pass-by-value}. The parameter (i.e. \texttt{x}) receives the value.

Here's a \textbf{very common gotcha}: If \texttt{main()} calls \texttt{square()} with a variable say \texttt{i}, and \texttt{x} (in \texttt{square()}) receives the \textbf{value} of \texttt{i}, then modifying \texttt{x} (in \texttt{square()}) does NOT affect the value of \texttt{i}.

Here's an example where the value of \texttt{i} is passed in to our \texttt{square()} function with \texttt{x} receiving the value. We change the value of \texttt{x} and back in \texttt{main()}, then value of \texttt{i} is not affected.

\begin{consolethree}[escapeinside=||]
#include <iostream>

int square(int x)
{
    int y = x * x;
    x = 0;
    return x * x;
}

int main()
{
    int i = 5;
    std::cout << square(i) << std::endl;
    std::cout << i << std::endl;

    return 0;
}
\end{consolethree}

Many students think that somehow the \textbf{variable} \texttt{i} (in \texttt{main()}) is passed in and that \texttt{x} (in \texttt{square()}) is somehow tied to the i (in \texttt{main()}).

That's \EMPHASIZE{NOT TRUE}. Only the value of \texttt{i} (in \texttt{main()}) is passed onto the \texttt{x} (in \texttt{square()}). \EMPHASIZE{The two variables have their own memory spaces. Changing one does not change the other.}

It's also important to remember that the memory of \texttt{square()} is created when you call it. When you return to \texttt{main()} (i.e. when you exit the block for \texttt{square()}), the memory for that particular call of \texttt{square()} is destroyed.

The following example shows you that when you call \texttt{square()}, the memory created for that call is different from the memory of \texttt{main()}. We'll create a variable \texttt{i} in \texttt{square()} and set it to zero. The \texttt{i} (in \texttt{main()}) is not changed.

\begin{consolethree}[escapeinside=||]
#include <iostream>

int square(int x)
{
    int y = x * x;
    int i = 0;
    return x * x;
}

int main()
{
    int i = 5;
    std::cout << square(i) << std::endl;
    std::cout << i << std::endl;

    return 0;
}
\end{consolethree}
