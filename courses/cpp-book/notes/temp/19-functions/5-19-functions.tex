\sectionthree{Passing in nothing}

You can also have a function that does not accept any value:

\begin{consolethree}[escapeinside=||]
#include <iostream>

void printHelloWorld()
{
    std::cout << "hello, world!" << std::endl;
    return;
}

int main()
{
    printHelloWorld();
    return 0;
}
\end{consolethree}

Of course you can have a function that accepts nothing but returns a value.

\begin{ex}
The function \texttt{f()} accepts nothing and returns a \texttt{char}. Write the function header of \texttt{f()}:

\end{consolethree}
\end{ex}
