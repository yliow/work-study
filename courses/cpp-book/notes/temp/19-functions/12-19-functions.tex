\sectionthree{Global scope}

Although the basic scope rules apply to functions (for instance you cannot access variables declared in one function from another), there is a ``super big'' scope that contains your entire code. This is called the \EMPHASIZE{global scope}.

\begin{consolethree}[escapeinside=||]
#include <iostream>

int y = 1;

void f()
{
    std::cout << y << '\n';
}

int main()
{
    std::cout << y << '\n';
    f();
    return 0;
}
\end{consolethree}
