Try to mix \texttt{std::cin} and \texttt{std::cin.getline()} like this:
\begin{consolethree}[escapeinside=||]
char s[100];
int i;
std::cin >> i;
std::cout << i << std::endl;
std::cin.getline(s, 100);
std::cout << s << std::endl;
\end{consolethree}

Run the program and see what happens.
