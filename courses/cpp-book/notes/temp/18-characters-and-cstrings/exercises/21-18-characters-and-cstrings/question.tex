\tinysidebar{\debug{exercises/{21-18-characters-and-cstrings/question.tex}}}Write a program accepts a string that is an integer expression, interprets the expression, perform the appropriate operations and prints the results. For instance is the user enters the string ``1 + 2'', the program prints 3. If the user enters ``1 - 2 + 3 * 4'' the program prints 11. The program should handle +, -, *, /, \% and integer values.
