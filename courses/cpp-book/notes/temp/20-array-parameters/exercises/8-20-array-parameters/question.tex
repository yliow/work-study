\begin{ex}
 Write a function \texttt{rand\_array()} that accepts an array of integers, the size of the array, min (an int), max (an int) and puts random integers from min, min+1, \ldots, max -- 1 into the array. For instance calling \texttt{rand\_array(x, 10, 1, 7)} will randomize x[0], x[1], x[2], \ldots, x[9] with integer values from 1 to 6. Test your function.
\end{ex}
