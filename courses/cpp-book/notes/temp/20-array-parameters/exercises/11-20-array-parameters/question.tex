\begin{ex}
Write a function \texttt{swap()} that accepts an array of integers and two index values, and swaps the values at those index positions. For instance if x is an array initialized with {1, 2, 3, 4}, then on return from calling \texttt{swap(x, 1, 3)}, the values in \texttt{x} becomes

1, 4, 3, 2

Test your function.
