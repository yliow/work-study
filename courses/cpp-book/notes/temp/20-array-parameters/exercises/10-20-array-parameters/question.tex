\begin{ex}
 Write a function \texttt{primes()} that accepts an array of integers and the size of the array and puts primes into the array starting with 2. For instance calling \texttt{primes(x, 5)} will set \texttt{x[0]} to 2, \texttt{x[1]} to 3, \texttt{x[2]} to 5, \texttt{x[3]} to 7, \texttt{x[4]} to 11. Test your function.
\end{ex}
