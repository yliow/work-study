\sectionthree{operator>> and operator<<}

You've been using \textit{<<} and
\textit{>>} since day 1 of CISS240. From your
previous notes you know that you can overload
\textit{operator>>} and
\textit{operator<<}. You also know from CISS240 that you
can output

\textit{int, char[], double}, etc., i.e., C++ has overloaded
\textit{operator>>} in iostream.h. You also know
that you can do

\textit{std::cout << "A" << 3 << 5.5;}

So the operators return an object.

\begin{consolethree}[escapeinside=||]
#include <iostream>

#ifndef INT_H
#define INT_H

class Int
{
public:
    Int(int x0 = 0) : x_(x0) {}
    int get() const { return x_; }
    ...
    void operator<<(std::ostream &, const Int &);
    ...
private:
    int x_;
};

#endif
\end{consolethree}

\begin{consolethree}[escapeinside=||]
#include "Int.h"

void operator<<(std::ostream & cout, const Int & c)
{
    cout << c.get();
}
\end{consolethree}

\begin{consolethree}[escapeinside=||]
#include "Int.h"

int main()
{
    Int c(42);
    std::cout << c;
    return 0;
}
\end{consolethree}
