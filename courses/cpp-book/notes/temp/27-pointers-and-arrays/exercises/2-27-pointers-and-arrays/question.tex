\tinysidebar{\debug{exercises/{2-27-pointers-and-arrays/question.tex}}}There are four pointer values (i.e., address values) in the following code segment. The first one is \texttt{p}. As for the rest:



\begin{itemize}

\item Is \texttt{p + 2} larger or smaller than \texttt{p}? How far apart are

  they?

\item Is \texttt{p - 1} larger or smaller than \texttt{p}? How far apart are

  they?

\item Is \texttt{p - 3} larger or smaller than \texttt{p}? How far apart are

  they?

\end{itemize}



\begin{consolethree}[escapeinside=||]

double d = 3.14;

double * p = &d;

std::cout << (unsigned int)(p) << ' '

          << (unsigned int)(p + 5) << ' '

          << (unsigned int)(p - 2) << ' '

          << (unsigned int)(p - 4) << '\n';

\end{consolethree}



Verify your answer by running the program.
