\sectionthree{Typename and class}

Instead of

\begin{consolethree}[escapeinside=||]
template  < |\textbf{typename}| X >
class C{}; 
\end{consolethree}

You can say

\begin{consolethree}[escapeinside=||]
template  < |\textbf{class}| X >
class C{}; 
\end{consolethree}

They mean the same thing. However, \texttt{typename} is better.
