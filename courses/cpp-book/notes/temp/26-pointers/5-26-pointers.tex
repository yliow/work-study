\sectionthree{Common Gotcha}

This is a very common gotcha:

\begin{consolethree}[escapeinside=||]
int x = 0, y = 1;
int * p = &x, q = &y, r = &x;
\end{consolethree}
It won't work (does not compile) because the above actually means this:

\begin{consolethree}[escapeinside=||]
int x = 0, y = 1;
int * p = \&x;
int q = \&y;
int r = \&x;
\end{consolethree}
It should be

\begin{consolethree}[escapeinside=||]
int x = 0, y = 1;
int * p = \&x, * q = \&y, * r = \&x;
\end{consolethree}
or (better) just write this:

\begin{consolethree}[escapeinside=||]
int x = 0, y = 1;
int * p = \&x;
int * q = \&y;
int * r = \&x;
\end{consolethree}
