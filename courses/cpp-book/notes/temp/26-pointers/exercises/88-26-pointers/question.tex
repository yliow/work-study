\tinysidebar{\debug{exercises/{88-26-pointers/question.tex}}}The following function \texttt{copy()} copies the data that one pointer is pointing to to the other pointer. In more details, \texttt{copy(p, q)} will



\begin{itemize}

\item

  Allocate memory for \texttt{p} (if \texttt{p} has not been allocated

  memory; \texttt{p} has value \texttt{NULL} if it has not been allocated

  memory)

\item

  Copy the integer \texttt{q} points to over to the integer \texttt{p}

  points to.

\item

  Deallocate the memory \texttt{q} points to and set \texttt{q} to

  \texttt{NULL}.

\end{itemize}

  Test it!
