\tinysidebar{\debug{exercises/{10-26-pointers/question.tex}}}In a code fragment, there are two variables \texttt{i} and \texttt{j}, both of \texttt{int} type. The value of \texttt{i} is 5000 and the value of \texttt{j} is 6000. The value of \texttt{i} is at address 108080 and the address of the value of \texttt{j} is 501200. In addition, there are two pointer variables \texttt{p} and \texttt{q} such that \texttt{p} points to the value of \texttt{j} and \texttt{q} points to the value of \texttt{i}. Draw the memory model labeling everything (variable name, value, and address). Include arrows to relate pointers to the values the pointers point to.
