\tinysidebar{\debug{exercises/{10-17-one-dimensional-arrays/question.tex}}}The following program computes the maximum value in variables a, b, c, d, e:



\begin{consolethree}[escapeinside=||]double a = 5.4;

double b = 1.2;

double c = 5.9;

double d = 3.14;

double e = 2.718;



double max = a;

if (max < b) max = b;

if (max < c) max = c;

if (max < d) max = d;

if (max < e) max = e; 

\end{consolethree}



We can use arrays instead:



\begin{consolethree}[escapeinside=||]

double a[] = {5.4, 1.2, 5.9, 3.14, 2.718};



double max = a[0];



for (int i = 1; i < 5; ++i)

{



    if (max < a[i]) max = a[i];

    {

        std::cout << max << std::endl;

    }

}

\end{consolethree}



If you don't see it, just unroll the for-loop.
