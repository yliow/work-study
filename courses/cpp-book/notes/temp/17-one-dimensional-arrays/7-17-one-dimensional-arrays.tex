\sectionthree{Sum of an array of values}

You can easily compute the sum of the values in an array:

\begin{consolethree}[escapeinside=||]
double a[] = {5.4, 1.2, 5.9, 3.14, 2.718};

double sum = 0.0;

for (int i = 0; i < 5; ++i)
{
    sum += a[i];
    std::cout << sum << std::endl;
} 
\end{consolethree}
