\begin{ex}
Write a function that accepts a 4-by-4 array of integers and returns true if for each row, the first non-zero value of each row is to the right of the previous one. For instance the following will return true:

\begin{align*}
1 & 3 & 3 & 4 \\
0 & 0 & 3 & 4 \\
0 & 0 & 0 & 2 \\
0 & 0 & 0 & 0 \\
\end{align*}
\end{ex}
