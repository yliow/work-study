\begin{ex}
Write a function that checks if there's a winning up-diagonal for character \texttt{player}:

\begin{consolethree}
bool has_winning_up_diag(char board[n][n], int c, char player);
\end{consolethree}

Character \texttt{player} is either \texttt{'X'} or \texttt{'O'}. The up-diagonal is the diagonal on the board that goes from row n-1, column 0 to row 0, column n - 1.

\begin{ex}
Write a function that checks if the board if filled (i.e. no spaces left):

\begin{consolethree}
bool is_filled(char board[n][n]);
\end{consolethree}

Character \texttt{player} is either \texttt{'X'} or \texttt{'O'}. The up-diagonal is the diagonal on the board that goes from row n-1, column 0 to row 0, column n - 1.

Make sure you test the above function thoroughly. Now complete the turn-based 2-player tic-tac-toe-game!
\end{ex}
