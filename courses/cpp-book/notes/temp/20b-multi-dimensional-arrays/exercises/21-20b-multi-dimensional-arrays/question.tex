\begin{ex}
Write a program that prompts the user to enter
his/her first name and then last name and stores them both in an array
called name. \texttt{name} should be declared like so:

\begin{consolethree}
char name[2][256]; 
\end{consolethree}

Print the name that they entered with a ``Hi'' -- see execution below.

\begin{consolethree}
What's you name? John Doe
  
Hi John Doe.
\end{consolethree}

\sectionthree{Exercise.}

Write a program that prompts the user for a sentence ending with a period of question mark. The program then prints the number of words entered by the user and list the words. Here's an execution:

\begin{consolethree}
Hi my name is John.

5 words

1. Hi

2. my

3. name

4. is

5. John 
\end{consolethree}

Here's another:

\begin{consolethree}
How are you?

3 words

1. How

2. are

3. you 
\end{consolethree}

The user can enter at most 100 words.
