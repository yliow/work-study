\begin{ex}
Write a function that accepts a 4-by-4 array of integers and returns true if and only if array as a matrix is symmetric. For instance, the following is symmetric:

\begin{align*}
1 & 3 & 3 & 4 \\
3 & 2 & 1 & 8 \\
3 & 1 & 0 & 2 \\
4 & 8 & 2 & 7 \\
\end{align*}

Think of the diagonal with values 1, 2, 0, 7 as a mirror. So the example is symmetric since for example the entry at row 0, column 3 is the same as the entry at row 3, column 0, i.e. 4.
\end{ex}
