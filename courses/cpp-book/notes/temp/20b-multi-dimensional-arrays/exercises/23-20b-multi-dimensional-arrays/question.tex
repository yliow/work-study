\begin{ex}
Prompt the user for an integer from 0 to 3 and print the corresponding T tetrad. There should only be one single triple for-loop (not 4). For instance if the user enters 0 the program should print:

\begin{consolethree}
X

XXX 
\end{consolethree}

If the user enters 1 the program should print

\begin{consolethree}
X

XX

X \\
\end{consolethree}
\end{ex}
