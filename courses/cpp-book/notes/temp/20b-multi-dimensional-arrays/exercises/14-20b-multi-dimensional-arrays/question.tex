\begin{ex}
Write a function to print the tic-tac-toe board:

\begin{consolethree}
...

void print(char board[n][n]);
\end{consolethree}

Suppose the board is filled with spaces and n is 3. Then the output is

\begin{consolethree}
+-+-+-+
| | | |
+-+-+-+
| | | |
+-+-+-+
| | | |
+-+-+-+ 
\end{consolethree}

If row 0 and column 1 of the board has `X', then the output is

\begin{consolethree}
+-+-+-+
| |X| |
+-+-+-+
| | | |
+-+-+-+
| | | |           
+-+-+-+
\end{consolethree}
\end{ex}
