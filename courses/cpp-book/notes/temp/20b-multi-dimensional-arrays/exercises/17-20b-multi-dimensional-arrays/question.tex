\begin{ex}
  Write a function that checks if there's a winning down-diagonal for character \texttt{player}:

\begin{consolethree}
bool has_winning_down_diag(char board[n][n], int c, char player); 
\end{consolethree}

Character \texttt{player} is either \texttt{'X'} or \texttt{'O'}. The down-diagonal is the diagonal on the board that goes from row 0, column 0 to row n - 1, column n -- 1.
\end{ex}
