\begin{ex}
It turns out that instead of

\begin{consolethree}[escapeinside=||]
...
g = &avg;
...
\end{consolethree}

you can also write

\begin{consolethree}[escapeinside=||]
...
g = avg;
...
\end{consolethree}

In other words, for the case of functions, the address operator \& is
optional. Try it.
\end{ex}
