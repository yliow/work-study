\tinysidebar{\debug{exercises/{1-28-pointers-and-functions/question.tex}}}It turns out that instead of



\begin{consolethree}[escapeinside=||]

...

g = &avg;

...

\end{consolethree}



you can also write



\begin{consolethree}[escapeinside=||]

...

g = avg;

...

\end{consolethree}



In other words, for the case of functions, the address operator \& is

optional. Try it.
