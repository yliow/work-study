\sectionthree{First sorting example}

Here's your first sorting problem:

\begin{tightlist}
\li
  Prompt the user for two integer values and assign them to x and y.
\li
  Sort the values so that the value of x is less than or equal to the
  value of y
\end{tightlist}

\begin{ex}
Write the above program. Here are some test cases:

Test 1
\begin{console}[commandchars=\~\@\$]
~textbf@1 2$
1 2
\end{console}

Test 2
\begin{console}[commandchars=\~\@\$]
~textbf@2 1$
1 2
\end{console}

Test 3
\begin{console}[commandchars=\~\@\$]
~textbf@-1 5$
-1 5
\end{console}

Test 4
\begin{console}[commandchars=\~\@\$]
~textbf@5 -1$
-1 5
\end{console}

Test 5
\begin{console}[commandchars=\~\@\$]
~textbf@2 2$
2 2
\end{console}
\end{ex}
