\tinysidebar{\debug{exercises/{23-61-constructors/question.tex}}}If \texttt{C} is a class and \texttt{obj1} is an object

of class \texttt{C} and you want to clone \texttt{obj1} to another object,

say \texttt{obj2}. The following works but it's a bad idea. Why?



\begin{consolethree}[escapeinside=||]

C obj2 = C(obj1);

\end{consolethree}



If you don't believe me you can try



\begin{consolethree}[escapeinside=||]

Date date1(1970, 1, 1);

Date date2 = Date(date1);

date2.print();

\end{consolethree}
