\sectionthree{Non-deducible context and ambiguity}

There are times when your C++ compiler won't be able to deduce the
correct type for your template parameter T.

Try to compile this:

\begin{consolethree}[escapeinside=||]
template < typename T >
T f(int x)
{
    return x;
}

int main()
{
    f(42);
    return 0;
}
\end{consolethree}

In this case, T can be \EMPHASIZE{int}, But note that it can also be
\EMPHASIZE{double} since

C++ allows automatic typecasting from \EMPHASIZE{int} to
\EMPHASIZE{double}.

In these cases, you have to explicit specify the substitution for
\EMPHASIZE{T}:

\begin{consolethree}[escapeinside=||]
template < typename T >
T f(int x)
{
    return x;
}

int main()
{
    f|\EMPHASIZE{< int >}|(42);
    return 0;
}
\end{consolethree}
