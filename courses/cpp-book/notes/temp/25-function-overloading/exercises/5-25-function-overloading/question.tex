Are there any ambiguous function calls? Find and circle if any.

\begin{consolethree}[escapeinside=||]
void f(int x, int y, int z)
{}

void f(int x)
{}

void f()
{}

void f(int x, int y, double z)
{}

int main()
{
    f(1);
    f(1, 2, 3);
    f(1.1, 3.3, 4.4);
    return 0;
}
\end{consolethree}

Verify with your C++.
