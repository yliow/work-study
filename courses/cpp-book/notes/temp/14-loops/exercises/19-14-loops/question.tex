\begin{ex} Review \ldots try this:
\begin{console}
std::cout << abs(5) << std::endl;
std::cout << abs(-5) << std::endl;
std::cout << fabs(12.34) << std::endl;
std::cout << fabs(-12.34) << std::endl; 
\end{console}

The pseudocode looks something like this:
\begin{console}
error = 0.00001
approx0 = [some value]
approx1 = [another value]
while (|approx0 -- approx1| >= error):
      approx1 = approx0
      compute approx1 
\end{console}
\end{ex}
