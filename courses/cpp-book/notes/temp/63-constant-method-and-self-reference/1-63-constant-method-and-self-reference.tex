\sectionthree{Recall}

Suppose we have a class \texttt{C} with a method \texttt{m}. Suppose an
object \texttt{obj} of \texttt{C} invokes \texttt{m}. In the body of method
\texttt{m}, members and member functions without objects are those
automatically associated with \texttt{obj}

\begin{consolethree}[escapeinside=||]
// Int.h
class Int |\xsidebox[0cm]{0cm}{}{recallbox0}{\textbf{When i invokes} \texttt{print()}, \textbf{the \texttt{x\_} refers to} \texttt{i.x\_}.}|
{
public:
    Int(int x) : x_(x) {}
    void print()
    {
        std::cout << |\tikzmarknode{recall0}{x\_}| << "\n";
    }
private:
    int x_;
};
\end{consolethree}
