\sectionthree{How to make \texttt{*this} constant?}

But what about \texttt{this}? It's passed in automatically and is not shown in the parameter list! How do you ``\texttt{const}'' \texttt{this}?

\begin{consolethree}[escapeinside=||]
class Int
{
public:
    Int(int x)
        : x_(x)
    {}
    Int max(|\EMPHASIZE{const}| Int & a)    |\tikzmarknode{a0}{}|
    {
        if (x_ < a.x_)|\xsidebox[0cm]{0cm}{}{abx0}{\texttt{a} is not changed so we made it constant}|
            return a;                    |\tikzmarknode{a1}{}|
        else
            return (*this);
    }
private:
    int x_;
};
\end{consolethree}
\DrawArrow{abx0}{a0}
\DrawArrow{abx0}{a1}

But ....

\begin{consolethree}[escapeinside=||]
class Int
{
public:
    Int(int x)
        : x_(x)
    {}
    Int max(|\EMPHASIZE{const}| Int & a)
    {
        if (x_ < a.x_)
            return a;
        else
            return (*this);|\tikzmarknode{a2}{} \xsidebox[0cm]{0cm}{}{abx1}{But \texttt{this} is also not changed. How do we make it constant?}|
    }
private:
    int x_;
};
\end{consolethree}
\DrawArrow{abx1}{a2}

Do this:

\begin{consolethree}[escapeinside=||]
class Int
{
public:
    Int(int x)
        : x_(x)
    {}
    Int max(|\EMPHASIZE{const}| Int & a) |\EMPHASIZE{\underline{const}} \tikzmarknode{a3}{}|
    {
        if (x_ < a.x_)
            return a;
        else
            return (*this);|\xsidebox[0cm]{0cm}{}{abx2}{\texttt{const} after function header will make \texttt{*this} constant in the method}|
    }
private:
    int x_;
};
\end{consolethree}
\DrawArrow{abx2}{a3}
Example:

\begin{consolethree}[escapeinside=||]
class Date
{
public:
    int get_d() |\EMPHASIZE{const;}|
private:
    int yyyy_, mm_, dd_;|\xsidebox[0cm]{0cm}{}{abx3}{Makes \texttt{*this} constant in the body of the function}|
};

int Date::get_d() |\EMPHASIZE{const}|
{
    return dd_;
}
\end{consolethree}

member functions}. In other words, constant member functions
are methods that will not modify the object invoking the method.

Should \texttt{operator*()} be constant? The question to ask is this: is
\texttt{p} changed? The important point ... There is a difference
between asking if \texttt{p} should be constant and if \texttt{*p} should be
constant.
