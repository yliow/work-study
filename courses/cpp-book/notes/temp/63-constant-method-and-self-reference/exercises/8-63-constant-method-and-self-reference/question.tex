\begin{ex}
Look at the \texttt{IntPointer} class very carefully.
In particular look at the de-reference operator, \texttt{operator*()}:

\begin{consolethree}[escapeinside=||]
class IntPointer
{
public:
    ...
    int & operator*()
    {
        return *p_;
    }
    ...
};
\end{consolethree}

Should \texttt{operator*()} be constant? The question to ask is this: is
\texttt{p} changed? The important point ... There is a difference
between asking if \texttt{p} should be constant and if \texttt{*p} should be
constant.
\end{ex}
