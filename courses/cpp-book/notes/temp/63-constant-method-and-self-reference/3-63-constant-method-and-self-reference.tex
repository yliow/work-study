\sectionthree{What is the Invoking Object}

What if a method must know the invoking object? Here's
such an example:

\begin{consolethree}[escapeinside=||]
class Int
{
public:
    ...
    |\textbf{Int max(const Int \& c)}|
    {
        if (c.x_ > x_)
        {
            return c;
        }
        else
        {
            return ???;
        }
    }
private:
    int x_;
};
\end{consolethree}

\begin{consolethree}[escapeinside=||]
int main()
{
    Int a(100), b(1);
    Int c = a.max(b);
    return 0;
}
\end{consolethree}

For \texttt{a.max(b)} we want to return \texttt{a}. But what's the name of
\texttt{a} in the code of \texttt{max()}??? In general, what's the name of
the object making the call?

There is a secret parameter in every method. You do not pass this
parameter into the method. It's done for you automatically.
