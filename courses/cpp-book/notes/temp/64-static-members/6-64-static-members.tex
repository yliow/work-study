\sectionthree{Application: Controlling default constructor using static object member}

Now let me give you a slightly more complicated example where the static
member is an object.

If you look at our Date class again, you recall that we use 1970/1/1 as
a default date in the default constructor:

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=1970, int=1, int=1);
    ...
private:
    int yyyy_, mm_, dd_;
};

#endif
\end{consolethree}

What if depending on the scenario of usage of the Date class, I
sometimes want to change the behavior of the default constructor during
runtime? In other words, occasionally, I might want to change the
default constructor to initialize the Date object something other than
1970/1/1. In other words I want this to happen in my main():

\begin{consolethree}[escapeinside=||]
#include "Date.h"

int main()
{
    Date date1; // date1 models 1970/1/1
    // do something here so that after this point
    // default Date constructor works differently
    Date date2; // date2 models 2000/12/25

    return 0;
}
\end{consolethree}

How would you do that??? For sure, your program (while
it's running) cannot possible change the default values
in the constructor!!!

Well ... here's I would do it ...

Instead of \textbf{hardcoding} 1970/1/1 as default values in the
constructor:

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=|{\bf 1970}|, int=|{\bf 1}|, int=|{\bf 1}|);
    ...
private:
    int yyyy_, mm_, dd_;
};

#endif
\end{consolethree}

I store 1970/1/1 as three static integers in the Date class:

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=1970, int=1, int=1);
    ...
private:
    ...
    |{\bf static int default\_yyyy\_;}|
    |{\bf static int default\_mm\_;}|
    |{\bf static int default\_dd\_;}|
};

#endif
\end{consolethree}

And initialize these static members in the cpp file:

\begin{consolethree}[escapeinside=||]
// Date.cpp
...
Date default_yyyy_(1970);
Date default_mm_(1);
Date default_dd_(1);
...
\end{consolethree}

These three default integer static members can be used by the default
constructor. For instance I can set the default values of the parameters
in the constructor to 0 to mean ``use the static default yyyy, mm, dd'':

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=|{\bf 0}|, int=|{\bf 0}|, int=|{\bf 0}|);
    ...
private:
    ...
    static int default_yyyy_;
    static int default_mm_;
    static int default_dd_;
};

#endif
\end{consolethree}

and in the \texttt{Date} implementation file I do this:

\begin{consolethree}[escapeinside=||]
//Date.cpp
#include "Date.h"

Date::Date(int yyyy, int mm, int dd)
{
    yyyy_ = (yyyy == 0 ? default_yyyy_ : yyyy);
    mm_ = (mm == 0 ? default_mm_ : mm);
    dd_ = (dd == 0 ? default_dd_ : dd);
}
...
\end{consolethree}

Furthermore I can write static methods to modify these static values.

Whoa ... wait ...

The three static integers in fact form a date ... why not just
bunch them up together into a default static \texttt{Date} object like
this:

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=0, int=0, int=0);
    ...
private:
    ...
    |{\bf static Date default\_date\_;}|
};

#endif
\end{consolethree}

and initialize \texttt{default\_date\_} in the cpp files like this:

\begin{consolethree}[escapeinside=||]
// Date.cpp
...
Date default_date_(1970, 1, 1);
...
\end{consolethree}

The constructor can now use this static default date like this:

\begin{consolethree}[escapeinside=||]
//Date.cpp
#include "Date.h"

Date::Date(int yyyy, int mm, int dd)
{
    yyyy_ = (yyyy == 0 ? default_date.yyyy_ : yyyy);
    mm_ = (mm == 0 ? default_date.mm_ : mm);
    dd_ = (dd == 0 ? default_date.dd_ : dd);
}
...
\end{consolethree}

And we can change the default date like this:

\begin{consolethree}[escapeinside=||]
// Date.h
#ifndef DATE_H
#define DATE_H

class Date
{
public:
    Date(int=0, int=0, int=0);
    ...
private:
    ...
    |{\bf static void set\_default\_date(}|
        |{\bf int, int, int);}|
};

#endif
\end{consolethree}

\begin{consolethree}[escapeinside=||]
//Date.cpp
#include "Date.h"

...

|{\bf void}| Date::set_default_date(int a, int b, int c)
{
    default_date_ = Date(a, b, c); // changing default
                                   // date
}
...
\end{consolethree}
