\begin{ex}
Create a public static \texttt{vec2d} object member in
your \texttt{vec2d} class so that you can run this:

\begin{consolethree}[escapeinside=||]
#include <iostream>
#include "vec2d.h"

int main()
{
    std::cout << vec2d::origin << '\n';

    return 0;
}
\end{consolethree}

Next, note that the member variables of \texttt{vec2d::origin} do not
change their values. Modify \texttt{vec2d::origin} so that it is a
\textbf{constant} public static member object of \texttt{vec2d}.
\end{ex}
