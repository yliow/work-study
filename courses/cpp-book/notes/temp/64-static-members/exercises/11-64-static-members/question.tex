for a computer game. The class is working well but you want to improve

on the performance to get a smoother animation. The game is very complex

and it's practically impossible to optimize every single

one of the 200 methods. But you still have 1 month before the game is to

be shipped and you think you can optimize one function. You want to

focus on the most frequently used method. You don't want

to read the code and manually count the number of times a method is

called. How would you figure out which method is used most?
