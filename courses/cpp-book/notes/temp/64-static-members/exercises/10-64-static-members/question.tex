\begin{ex}
In your \texttt{Date} class, keep a static \texttt{Date} object
\texttt{earliest\_date} so that the earliest \texttt{Date} object
instantiated will have the same values as \texttt{earliest\_date}. If the

\texttt{earliest\_date}, then the value of \texttt{earliest\_date} is used
instead.
