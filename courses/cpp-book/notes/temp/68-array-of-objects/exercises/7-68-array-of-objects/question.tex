Hint: Note that

\begin{console}

a[1][2]

\end{console}          

Is not \texttt{a.operator[][](1,2)} because there is no

\texttt{[][]} operator!!! Instead \texttt{a[1][2]} is

\texttt{a.operator[](1).operator[](2)}
