Pick another CFG and convert it to a PDA that accepts the same language that is
generated by the CFG. 
For instance you can try
\begin{tightlist}
  \item[(a)] $\{ww^R \st w \in \{a, b\}^*\}$, 
  \item[(b)] $\{a^n b^{2n} \st n \geq 0 \}$, etc.
\end{tightlist}
