Let $L_1 = \{a^n \mid n \geq 0\}$ and $L_2 = \{ab^na \mid n \geq 0\}$.
Which of the following strings are in $L_1 L_2$?
(Note that in this case the languages are infinite.)
\begin{tightlist}
\item $\ep$
\item $a$
\item $b$
\item $aa$
\item $ab$
\item $ba$
\item $bb$
\item $aaa$
\item $aab$
\item $aba$
\item $baa$
\item $abb$
\item $bab$
\item $bba$
\item $bbb$
\end{tightlist}
