   Prove or disprove the following:
   Let $m,n \geq 0$ be integers.
   \begin{tightlist}
   \item $x^{n + 1} = x x^n$
   \item $(x^m)(x^n) = x^{m+n}$
   \item $(x^m)(x^n) = (x^n)(x^m)$
   \item $(x^m)^n = x^{mn}$
   \item $(xy)^m = x^m y^m$
   \end{tightlist}
