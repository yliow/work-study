What is the big-O of the following functions.
Always choose the simplest and smallest.
\begin{enumerate}[nosep, label=(\alph*)]
  \item $100$
  \item $5n^2 - 10$
  \item $42 - 3n$
  \item $5 - n + n^3 + 2n^2$
  \item $5 - n + 1000000n^3 + n^2$
\end{enumerate}
