Let
\[
f(n) = 123456789 + (-1)^n n^3
\]
Using plots, 
find the smallest \textit{real number} $k$ (not necessarily an integer) 
such that $f(n) = O(n^k)$.
Supply a $C$ and an $N$ such that 
\[
|f(n)| \leq C |g(n)|
\]
for $n \geq N$.
