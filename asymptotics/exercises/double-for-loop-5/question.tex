Here's another algorithm:
\begin{Verbatim}[frame=single, fontsize=\small]
s = 0
for i = 0, 1, 2, ..., n - 1:
    for j = 0, 1, 2, ..., i:
        for k = j, j + 1, ..., n - 1:
            s = s + x[i][j] + k
\end{Verbatim}
Compute the big-O of the runtime of this algorithm.
