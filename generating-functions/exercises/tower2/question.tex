Let $a_n$ ($n=0, 1, 2, \ldots$) satisfy
\[
a_n = 
\begin{cases}
1 &\text{ if } n = 0 \\
2 &\text{ if } n = 1 \\
a_{n-1} + a_{n-2} + 3 &\text{ if } n > 1
\end{cases}
\]
Let $a(x) = \sum_{n=0}^\infty a_n x^n$.
Find a rational function for $a(x)$.
Find a closed form for $a_n$.
