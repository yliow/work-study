We have
\begin{align*}
(n+1)^2 = n^2 + 2n + 1
\end{align*}
Hence
\[
a_{n + 1} = a_n + 2n + 1
\]
or
\[
a_{n} = a_{n - 1} + 2(n - 1) + 1 = a_{n - 1} + 2n - 1
\]
for $n \geq 1$.
Hence
\[
a_n =
\begin{cases}
  0                 & \text{ if } n = 0 \\
  a_{n - 1} + 2n - 1 & \text{ if } n > 0 
\end{cases}
\]
This is a nonhomogeneous linear recurrence of degree 1.
\qed
