\starprob
Let $\phi$ be the Euler $\phi$--function.
This defines a sequence
$\phi(1)$, 
$\phi(2)$, 
$\phi(3)$, ...
Does this sequence satisfy a recurrence relation?
If you can find one, is that linear? homogeneous? What is the degree?
