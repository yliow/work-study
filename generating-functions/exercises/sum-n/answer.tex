Note that
\begin{align*}
s_n &= 0 + 1 + 2 + \cdots + n \\
    &= s_{n - 1} + n
\end{align*}
for $n \geq 1$.
Clearly $s_0 = 0$.
This is a nonhomogeneous linear recurrence relation of degree 1.
Here's a complete description of $s_n$:
\[
s_n =
\begin{cases}
  0 & \text{ if } n = 0 \\
  s_{n - 1} + n & \text{ f } n \geq 1
\end{cases}
\]
\qed
