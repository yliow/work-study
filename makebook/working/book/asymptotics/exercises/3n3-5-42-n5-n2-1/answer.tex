If we choose $C = 4$ and $N = 37$,
we see that for $n \geq N$,
\[
\left|
-3n^{3.5} + 42 \frac{n^5}{n^2 + 1}
\right| 
\leq C
\left|
n^4
\right|
\]
i.e.,
\[
|f(n)| \leq C|g(n)|
\]
Hence
\[
f(n) = O(n^4)
\]
\qed
