(a)
The worst case runtime of the given bubblesort is
\[
O(n^2 \times \text{time to execute \texttt{swap}})
\]
The runtime of \verb!swap! is $O(n)$.
Therefore the worst case runtime of the given bubblesort is
$T(n) = O(n^3)$.
