If we choose $C = 1$ and $N = 1600$,
we see that for $n \geq N$,
\[
\left|
0.31415 n^{4} + 1000n^3 
\right| 
\leq C
\left|
n^4
\right|
\]
i.e.,
\[
|f(n)| \leq C|g(n)|
\]
Hence
\[
f(n) = O(n^4)
\]
\qed
