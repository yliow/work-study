If we choose $C = 8$ and $N = 3$,
we see that for $n \geq N$,
\[
\left|
7n^{4} + 20
\right| 
\leq C
\left|
n^4
\right|
\]
i.e.,
\[
|f(n)| \leq C|g(n)|
\]
Hence
\[
f(n) = O(n^4)
\]
\qed
