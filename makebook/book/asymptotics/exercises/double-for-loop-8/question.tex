Yet another algorithm:
\begin{Verbatim}[frame=single, fontsize=\small]
s = 0
for i = 0, 1, 2, ..., (n - 1)/2:
    for j = 0, 1, 2, ..., n/4:
        for k = i, i + 1, ..., n - 1:
            s = s + x[i][k] * j
\end{Verbatim}
Compute the big-O of the runtime of this algorithm.
