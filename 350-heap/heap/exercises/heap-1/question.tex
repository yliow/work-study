Suppose you have this array:
\[
5, 4, 3, 2, 1
\]
(index 0 is on the left).
Assume that this array represents a binary tree.
\begin{tightlist}
  \item Is this binary tree a maxheap?
  \item How many arrays with the above values represent a maxheap?
\end{tightlist}
